동양미래대학교 반도체부트캠프 사용자 중심 개선 시안

실행 방법
1. ZIP 파일 압축 해제
2. index.html 더블클릭
3. 상단 메뉴에서 교육과정, 신청방법, 실습장비, 공지·후기, 채용정보, FAQ 페이지로 이동

파일 구성
- index.html: 메인 화면
- courses.html: 교육과정 안내 및 검색/필터
- apply.html: 신청방법
- facility.html: 실습장비
- community.html: 공지사항 및 수강후기
- jobs.html: 채용정보
- faq.html: 자주 묻는 질문
- style.css: 전체 디자인
- script.js: 모바일 메뉴, 교육과정 필터, 교육과정 탭 기능
